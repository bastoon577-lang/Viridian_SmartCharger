# Viridian_SmartCharger

That project operates in collaboration with [SmartCharger_Utils](https://github.com/bastoon577-lang/SmartCharger_utils) and a
allows the integration of SmartCharger code through the use of [Viridian system](https://www.amazon.fr/Viridian-EV-EPC20L10-Contr%C3%B4leur-Blanc/dp/B097BTRGYL).

For more information, consult the [Wiki](https://github.com/bastoon577-lang/Viridian_SmartCharger/wiki).

## Quick Install repository
```bash
git clone https://github.com/bastoon577-lang/Viridian_SmartCharger.git
cd Viridian_SmartCharger/SOFTWARE/Viridian_SmartCharger/
git submodule update --init --recursive
```
